# Tic Tac Toe

A modern, visually polished Tic Tac Toe game playable in any browser.

## Play Online

**https://viswakanth98.github.io/tictactoe/**

No install, no login — just open and play.

## How to Play

1. The game is **2 players on the same device** — Player X goes first.
2. Take turns clicking a cell to place your symbol.
3. First to get **3 in a row** (horizontally, vertically, or diagonally) wins.
4. If all 9 cells are filled with no winner, it's a **draw**.
5. Click **NEW GAME** to reset the board. Scores carry over between rounds.

## Run Locally

Download or clone the repo, then open `index.html` in any browser:

```bash
git clone https://github.com/viswakanth98/tictactoe.git
cd tictactoe
open index.html
```
